WHAT IS C++ ?

C++:

C++ is an extension of the older language C, with added features like Object-Oriented Programming (OOP). It was developed by Bjarne Stroustrup in the early 1980s.

🔹 Key Features of C++

🔹 Object-Oriented


C++ lets you organize code using classes and objects.

Class = blueprint
Object = real-world instance

EXAMPLE :

class Car { public: string color; cout << "Car is driving"; } };

🔹Fast and Efficient

C++ is close to hardware, so programs run very fast compared to languages like Python.

🔹 Compiled Language

C++ code is converted into machine code using a compiler before running.

🔹 Supports Multiple Programming Styles

Procedural (like C)

Object-Oriented

Generic (templates)

🔹 Memory Contro

You can manually control memory using pointers:
int* ptr = new int(10); delete ptr;
